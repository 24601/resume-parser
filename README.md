# Resume Parser

A Simple NodeJs library to parse Resume / CV to JSON.

## Usage

Install using

`yarn install resume-parser`

Parse resume:

```
const ResumeParser = require('resume-parser');
 //input file, output dir
ResumeParser.parseResume('./files/resume.doc', './files/compiled');
```
